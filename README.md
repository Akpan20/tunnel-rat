A cross renderer portal that allows you to project from one place into another. Demo: https://codesandbox.io/s/basic-demo-forked-kxq8g

```jsx
import tunnel from 'tunnel-rat'

const t = tunnel()
// <t.In>{content}</t.In>
// <t.Out />
```
